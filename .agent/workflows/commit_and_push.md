---
description: Stage, commit, and push all changes to GitHub
---

1. Check current git status
   ```bash
   git status
   ```

2. Stage all changes
   // turbo
   ```bash
   git add .
   ```

3. Commit changes (IMPORTANT: You must provide a commit message)
   ```bash
   git commit -m "update"
   ```
   > [!IMPORTANT]  
   > **Please replace "update" with a descriptive commit message before running this step.**

4. Push to remote
   // turbo
   ```bash
   git push origin main
   ```
